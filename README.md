# pscf - Processing standardized configuration file.

## Introduction
PSCF tools provide configuration file add, delete, and get operations.

### Supported configuration file formats

* YAML
* JSON
* INI

## Installation
`go get  github.com/liupeidong0620/pscf`

## Compile
```
linux
make build-linux

MacOS
make build-darwin

winOS
make build-windows
```
